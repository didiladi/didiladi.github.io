Projekte

Dies sind meine persönlichen Projekte, die ich hauptsächlich verfolge, um neue Technologien kennenzulernen. Generell bin ich eine rastlose Person, die ständig versucht ihr Wissen zu erweitern.

Photo Pilot

Ever wondered if the cumbersome task of manual photo selection (for e.g. a photo album) could be automated? Photo Pilot to the rescue! This project is a platform independent Electron app which uses convolutional neural networks (CNN) to decide whether a photo fulfills certain quality standards.

Fitness Tracker

How to build your own machine learning-enabled fitness tracker? This personal project explores several topics, from microcontroller and sensor hardware, firmware development and machine learning for motion detection.

Expense Phoenix

Expense Phoenix is an Android/iOS app for tracking your personal expenses. The goal was to create a platform-independent ionic app which can be trusted. Sure, there are a ton of other apps out there, which provide the same functionality. However, can you really trust these apps with your personal financial data?

Color Swipe

Color Swipe is a thrilling Android/iOS game where you have to swipe the colors into the correct direction. It is based on ionic which sits on top of apache cordova. The UI is done mainly with d3.
