Color Swipe

Sat, Nov 5, 2016

Color Swipe is a thrilling Android/iOS game where you have to swipe the colors into the correct direction. It is based on ionic which sits on top of apache cordova. The UI is done mainly with d3.

I’m planning to release the source of the app as open source. Stay tuned for more information.
